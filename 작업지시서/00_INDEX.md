# 작업지시서 (Work Orders)

Stock Agent Rev 프로젝트의 **작업지시서** 모음. 신규 작업은 `WO-순번_제목.md`로 이 폴더에 누적.

## 작성 규칙
- 한 WO = 한 워크스트림. 각 WO: 목적 / As-Is / To-Be / 구현 / 판정·기준 / 검증 / 영향 파일.
- **실제 코드 수정은 Codex가 repo에서 수행**, 본 문서는 그 스펙(지시).
