# WO-003 · 스코어링 확장 — 세력 유입(%) + 종합점수

목적: 종목별 **세력(큰손) 유입 정도를 0~100%로 산출**하고, 기존 차트점수와 합쳐 **차트/세력/종합 3종 점수**를 제공.
데이터 정책(확정): **실데이터(기관·외국인 순매수) 우선, 실패 시 OHLCV 프록시 폴백.**
선행 의존: WO-001 P0-1(캐시). 자동 실행/버튼은 WO-004.

> 면책: "세력 유입"을 100% 확인하는 신호는 없다. 차트·거래량·수급의 **비정상 매집 흔적을 확률적으로** 읽는 것. 매매 권유 아님. 임계값·가중치는 백테스트로 보정.

> 설계 핵심(개정): 같은 신호도 **위치(바닥/고점)에 따라 의미가 뒤집힌다**(바닥 대량=매집, 고점 대량=분산). 따라서 점수는 "+신호 가중합"이 아니라 **① 매집 가점 → ② 위치 가중 → ③ 분산 감점 → ④ 시퀀스 보너스** 순으로 계산한다.

---

## 1. 세력유입 점수 (0~100%) — 계산 구조

```
raw      = Σ(가중치 × 매집서브점수)              # 0~100, 아래 A(실데이터) 또는 B(프록시)
ctx      = position_multiplier (0.6 ~ 1.2)        # 위치 가중(바닥권↑ / 고점권↓)
penalty  = Σ(분산 감점)                            # 0~40, 고점 대량음봉 등
seq      = sequence_bonus (0 ~ 15)                # 돌파→지지→눌림→재상승 단계 통과
force_inflow_pct = clip( raw*ctx - penalty + seq , 0, 100 )  # round 1
```
경로 표기 `force_inflow_source` = real / proxy / mixed.

등급 `force_inflow_grade`: ≥75 강한유입 / 60~75 유입우세 / 40~60 중립 / 25~40 약함 / <25 이탈·분산.

---

## 2. 매집 가점 — 경로 A: 실데이터 (pykrx, 우선)

소스: `stock.get_market_trading_value_by_date`(외국인/기관/개인 순매수대금), 외국인 보유비중, 시총/거래대금. 윈도우 20거래일.

| # | 신호 | 정의 | 정규화 | 가중 |
|---|------|------|--------|------|
| A1 | 외국인 순매수강도 | Σ외국인순매수 ÷ Σ거래대금(20d) | x/0.15 | 0.24 |
| A2 | 기관 순매수강도 | Σ기관순매수 ÷ Σ거래대금 | x/0.15 | 0.18 |
| A3 | 외국인 연속순매수일 | 연속일수 | days/7 | 0.13 |
| A5 | 외국인 지분율변화 | Δ보유비중(%p) | Δ/2.0 | 0.13 |
| A4 | 기관 연속순매수일 | 연속일수 | days/7 | 0.08 |
| A6 | 순매수 가속 | 최근5일합 ÷ 직전15일 일평균 | (r-1)/2 | 0.08 |
| A7 | **개인 비편중** | 1 − 개인순매수/(외국인+기관+개인 절대합) | clip0~1 | 0.08 |
| A8 | **건강한 상승**(신용) | 신용잔고 급증 없이 상승 | 급증無&상승 1 / 보통 0.5 / 급증 0 | 0.08 |

- 음수(순매도)는 0 클립.
- A8 신용잔고는 **데이터 가용 시**(pykrx/별도 소스). 못 받으면 A8 제외하고 나머지 가중 재정규화.
- "개인만 대량매수"(A7 낮음)는 추격과열 신호 → 가점 축소.

## 3. 매집 가점 — 경로 B: OHLCV 프록시 (실데이터 실패 시)

보조지표: OBV, CMF(20), 매집봉(거래량≥1.5×평균 & 종가 레인지 상위60%↑) 비율.

| # | 신호 | 정의 | 정규화 | 가중 |
|---|------|------|--------|------|
| B1 | CMF(20) | 자금흐름 | CMF/0.15(음수→0) | 0.20 |
| B2 | OBV 추세 | OBV 20일 회귀기울기/평균거래량 | slope/0.5 | 0.16 |
| B9 | **상승/하락 거래량 비대칭** | 상승일 평균거래량 ÷ 하락일 평균거래량 | (r-1)/1.0 | 0.18 |
| B10 | **하락 시 거래량 감소** | 최근 하락일 거래량 ÷ 직전 하락일 거래량 | (1-r)/0.4 | 0.10 |
| B4 | 매집봉 비율 | 일수/20 | ratio/0.4 | 0.12 |
| B3 | 거래량 급증 | 최근5일 ÷ 직전20일 평균 | (r-1)/1.5 | 0.10 |
| B11 | **박스권 돌파+지지** | N일 저항 종가 돌파 & 돌파가 위 유지 | 돌파+지지 1 / 돌파만(종가밀림) 0.3 / 무 0 | 0.08 |
| B6 | 횡보 매집 | ATR%↓ + 거래량↑ 동시 | 1/0.5/0 | 0.06 |

- B9가 핵심(바닥 매집의 정석: 오를 땐 거래량 붙고 내릴 땐 마름). B11은 `vcp.py`/차트분류 돌파로직 재사용. **장중 돌파·종가 밀림은 가짜로 0.3만**.

### mixed
- 실데이터 일부만 확보 시 확보분 + 프록시 보완, `source="mixed"`.

## 4. 위치 가중 (position_multiplier) — 같은 신호, 위치 따라 의미 반전

기준: 현재가의 위치(예: 52주 레인지 백분위 `pctile`, MA200·베이스 대비, 최근 상승폭).

| 위치 | 판정 | 배수 |
|------|------|------|
| 바닥권/베이스(저점 상승 + 횡보) | 매집 유력 구간 | **1.2** |
| 중립(추세 초중반) | — | 1.0 |
| 단기 급등 직후(고RSI·5일 급등) | 과열 | 0.8 |
| 장기 상승 후 고점권(52주 상위 + 장기 +) | 분산 위험 | **0.6** |

> "윗꼬리 반복 + 저점 상승 + 거래량 증가"는 **바닥권에서만** 물량테스트로 가점, 고점권에선 분산으로 본다(아래 패널티와 연동).

---

## 5. 분산(나가는) 감점 (penalty, 0~40)

고점 대량 = 매집 아니라 매도일 확률↑. **명시적 마이너스**.

| 신호 | 조건 | 감점 |
|------|------|------|
| 고점권 대량 장대음봉 | 위치 고점 & 거래량≥2×평균 & 큰 음봉 | −15 |
| 급등 후 거래량 안 줄고 하락 | 최근 하락인데 거래량 유지/증가 | −10 |
| 고점권 윗꼬리 반복 | 위치 고점 & 윗꼬리 클러스터 | −8 |
| 돌파 실패(다음날 반납) | 대량양봉 익일 상승분 대부분 반납 | −7 |
| 거래량 폭발 + 종가 시가 아래 | 대량인데 종가<시가 | −5 |

- 패널티는 위치가 바닥권이면 약화(× 0.5), 고점권이면 강화(× 1.0).

---

## 6. 시퀀스 보너스 (sequence_bonus, 0~15)

스칼라 합산이 못 보는 **시간 순서**를 단계 통과로 가점.
단계: ① 바닥권 횡보 → ② 거래량 서서히 증가 → ③ 하락 시 거래량 감소 → ④ 박스 상단 거래량 돌파 → ⑤ 돌파가 지지 → ⑥ 다음날 거래량 줄며 눌림 → ⑦ 재상승.
- 연속 통과 단계 수에 비례(예: 통과 단계당 +2.5, 최대 15). ⑤+⑥ 동시 충족 시 가산.

---

## 7. 종합점수 (차트 × 세력)

```
chart_score     = adjusted_score(없으면 final_score)
force_score     = force_inflow_pct
composite_score = round(w_chart×chart_score + w_force×force_score, 1)   # 0~100
기본 w_chart 0.6 / w_force 0.4
```
- 세력 결측 종목: `composite_score = chart_score`, `composite_source="chart_only"`(기본) 또는 N/A.
- (선택) `mode: gate`: force < gate_min_force면 패널티(×0.8)/제외.

---

## 8. 자동 제외 (자동화 부적합 → 수동 체크리스트로 분리)

데이터 신뢰도/가용성 문제로 **점수에 넣지 않음**(별도 안내 텍스트로만 제공):
- 호가창/체결강도/허매수·매도벽 — 일봉 데이터에 없음 + 조작 多.
- 종토방·텔레그램 언급 폭증 — 소셜 피드 필요, 범위 밖.
- 내부자·대주주 매도 공시 — 추후 DART 연동 시 "위험 플래그"로 별도(점수 외).

UI "점수 기준" 탭(WO-004)에 위 수동 체크 항목을 참고 체크리스트로 표시.

## 10. 신규 모듈 `src/force_inflow.py`

## 12. 판정·검증
- [ ] **고점권 대량거래 종목이 세력점수 높게 안 나옴**(위치 가중·분산 감점 동작) — 핵심 회귀.
- [ ] 바닥권 "상승거래량>하락거래량 + 하락 시 거래량 마름 + 박스 돌파 후 지지" 종목이 상위.
- [ ] 장중 돌파·종가 밀림은 박스돌파 점수 0.3만(가짜 구분).
- [ ] 과거 외국인·기관 강매수 사례 real 경로 상위(스폿체크). pykrx 실패 시 proxy 폴백.
- [ ] 종합점수 = 가중합 일치, 세력 결측은 chart_only. 캐시 적중 시 빠름.
