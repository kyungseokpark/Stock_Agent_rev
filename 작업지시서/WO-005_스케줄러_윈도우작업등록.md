# WO-005 · 스케줄러 — 추천 시간 자동 + 윈도우 작업 스케줄러 원클릭 등록

목적: 컴퓨터가 켜져 있으면 **추천 시간에 자동 스크리닝**, **버튼 하나로 윈도우 작업 스케줄러 등록**.
환경: 본인=**한국(KST, UTC+9)**, 아버지=**인도네시아 자카르타(WIB, UTC+7)** — 2시간 차. 새벽(수면) 실행 회피가 핵심.
선행 의존: WO-001 P0-1(캐시), WO-004 C-1(자동 세력/종합). 선택: `telegram_sender.py` 연동.

---

## 1. 추천 시간 설계

장 시간: 국장 09:00–15:30 KST(마감 후 일봉 확정). 미국장 09:30–16:00 ET →
- EDT(여름): 16:00 ET = **05:00 KST / 03:00 WIB**
- EST(겨울): 16:00 ET = **06:00 KST / 04:00 WIB** → **새벽**.

**원칙: 미국장은 "마감 직후"가 아니라 "다음 날 아침"에 실행.** 일봉은 마감 후 고정 → 아침 실행으로 데이터 충분 + 수면 방해 없음 + **미 DST 영향 무시 가능**(아침 고정시각).

### 추천 기본값(평일 Mon–Fri)
| 작업 | 한국(KST) | 인니(WIB) | 근거 |
|------|-----------|-----------|------|
| 국장 브리핑 | **16:00** | **14:00** | 코스피/코스닥 15:30 마감 직후(동일 시점) |
| 미국장 브리핑 | **08:00** | **07:30** | 전일 미 마감(새벽) 후 아침, 수면 회피 |
| 둘 다 | 08:00 미국 + 16:00 국장 | 07:30 미국 + 14:00 국장 | 아침=미국, 오후=국장 |

- 주말/휴장일: 실행돼도 새 데이터 없음 → "휴장" graceful 처리(에러 금지). 평일만 등록.
- 추천값 프리필, 사용자 수정 가능.

## 2. 헤드리스 자동 실행
- `python main.py --market kr|us [--auto] [--notify telegram]` (`--auto` 무인 실행 후 종료, `--notify` 결과 텔레그램 발송 = 아버지 폰 수신). "둘 다"는 작업 2개 또는 `--market both`.
- EXE 동일 인자: `StockAgentRev.exe --market kr --auto`.

---

## 3. 윈도우 작업 스케줄러 원클릭 등록

`schtasks`(사용자 작업, 비번 불필요 / "로그온 시 실행"):
```bat
schtasks /Create /TN "StockAgent\KR_1600" /SC WEEKLY /D MON,TUE,WED,THU,FRI /ST 16:00 ^
  /TR "\"C:\path\StockAgentRev.exe\" --market kr --auto" /RL LIMITED /F
schtasks /Create /TN "StockAgent\US_0800" /SC WEEKLY /D MON,TUE,WED,THU,FRI /ST 08:00 ^
  /TR "\"C:\path\StockAgentRev.exe\" --market us --auto" /RL LIMITED /F
```
- 해제 `/Delete /TN ... /F` · 조회 `/Query /TN ...`.
- 권장 조건(정밀 설정은 `/XML` import): ✅컴퓨터 깨우지 않음 ✅놓친 작업 가능한 빨리 보충 ✅AC/배터리 무관 ✅로그온 시 실행(비번 불필요).

### 구현
- 신규 `src/scheduler.py`: `recommend_times(profile)`(KR/ID 표 반환), `build_schtasks_command(...)`, `register_task/unregister_task/list_tasks`(`subprocess`). 경로 자동 탐지(`sys.executable` 또는 `python main.py`).
- UI 신규 **"스케줄" 카드/탭**:
  - 프로필 라디오: **본인(한국/KST) / 아버지(인도네시아/WIB)**.
  - 추천시각 프리필(수정 가능): 국장/미국장.
  - 체크: ☑국장 ☑미국장(또는 둘 다), ☑텔레그램 수신.
  - 버튼: **[윈도우 스케줄러에 등록] / [등록 해제] / [상태 확인]**.
  - 상태 라벨 "등록됨: KR 16:00 / US 08:00 (평일)". 성공/실패 messagebox.

- [ ] [해제] 동작. 주말/휴장 에러 없이 "데이터 없음".

## 6. 비고
- 작업 스케줄러는 **PC 켜져 있을 때만** 동작(요건 일치). 24h 자동은 별도 상시구동 필요(범위 밖).
- 아버지 PC는 시스템 시계가 WIB → `/ST`가 현지시각 기준 등록되므로 "인도네시아" 프로필로 등록하면 의도대로.
- 미 DST 전환에도 아침 고정 실행이라 재조정 불필요.
